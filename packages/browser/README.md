# packages/browser

## Description

Frontend-oriented shared packages:

- `Authenticator`: login/sign-up form components
- `Toast`: notification provider/components
- `ui`: core shared UI component system

## Capabilities & Exports

- UI foundation exports: shared design-system components, theme primitives, and iconography consumed by browser applications.
- Authentication UI exports: reusable login/sign-up form components with validation and typed payload support.
- Notification exports: toast provider/context and presentation components for cross-app feedback UX.
