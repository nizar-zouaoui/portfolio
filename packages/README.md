# packages

## Description

Reusable monorepo packages grouped by concern:

- `browser/`: React UI/auth/toast packages
- `node/`: backend middleware + shared type utilities
- `SDK/`: typed API client layers
- `configs/`: ESLint/TypeScript/Tailwind shared configuration packages

## Capabilities & Exports

- Browser package family: exports reusable UI primitives, authentication forms, and toast notification infrastructure for frontend apps.
- Node package family: exports backend middleware and shared contract types used by APIs and SDK layers.
- SDK package family: exports typed API clients and base transport abstractions used by app runtimes.
- Config package family: exports shared linting, type-checking, and design-token standards for all workspaces.
